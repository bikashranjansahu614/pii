# Security Rules

Every request must be authenticated.

Every request must be authorized.

No raw PII reaches an LLM.

PII detection always precedes tokenization.

Policy evaluation always precedes LLM access.

Responses must always be validated.

Every request must generate an audit record.

Secrets must only come from Azure Key Vault.

No hardcoded credentials.

No plaintext secrets.

TLS everywhere.

AES-256 for sensitive data.
